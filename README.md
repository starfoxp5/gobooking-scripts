# gobooking-scripts

GoBooking 場地預約自動化腳本集（Fiona Agent 專用）

## 腳本
- **gobooking_playwright.py** - 預約球場核心引擎
- **fiona_book_court.py** - 預約球場 CLI
- **gobooking_cancel.py** - 取消預約
- **gobooking_cancel_sync.py** - 取消同步 Google Calendar
- **skill_loader.py** - Fiona skill 定義

## 其他工具
- **gobooking_reschedule.py** - 更改預約
- **gobooking_check_court.py** - 查詢球場狀態
- **gobooking_book_k.py** - 快速預約
- **gobooking_jq_batch.py** - 批量操作
- **gobooking_ticket_status.py** - 查詢票券狀態

## 用法
見個別腳本的 docstring 或執行 `--help`
